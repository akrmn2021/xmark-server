Firefox extension (first beta version 1.2)
Chrome extension - coming soon
Github repo for xmark-client: github.com/akrmn2021/xmark-client
Github repo for xmark-server: github.com/akrmn2021/xmark-server

Note: The extension is in the process of being signed by Mozilla. Until then, to use it, you must:
Create a directory for the extension
Place the zip file in the directory
Extract the zip file
Go to about:debugging in Firefox
Click on 'This Firefox' in the top left menu
Load Temporary Add-on...
Select any file from the extension's directory, such as manifest.json
